# LibraFlow

LibraFlow est une application complète avec frontend, backend et module IA.

## Structure du projet

- **frontend/**: Application frontend
- **backend/**: API et logique backend
- **ia/**: Module Intelligence Artificielle

## Installation

```bash
# Frontend
cd frontend
npm install

# Backend
cd ../backend
npm install

# IA
cd ../ia
pip install -r requirements.txt
```
